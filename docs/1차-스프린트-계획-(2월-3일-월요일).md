## 백엔드 개발 계획

프론트엔드 개발 일정에 맞춰 기본적인 CRUD API 구현에 집중하기로 했습니다. 프론트엔드에서 필요한 순서대로 API를 우선 개발하여 프론트엔드 작업이 막히지 않도록 진행할 예정입니다.

## 프론트엔드 개발 계획

**UI 개발**
- 이번 주 내로 모든 UI 작업을 완료하는 것을 목표로 설정했습니다.

**API 연동**
다음 API들을 우선순위에 따라 연동할 예정입니다:

- **인증 관련**: 회원가입, 로그인
- **자기소개서 관리**: 자기소개서 리스트 조회, 상세 정보 조회, 저장
- **검색 기능**: 자기소개서 전체 검색, 라이브러리 내 기업/문항 검색
- **라이브러리**: 문항 스크랩
- **메인 화면 및 캘린더**: 메인 화면 데이터, 캘린더 데이터 연동

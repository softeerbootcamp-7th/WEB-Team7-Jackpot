### 1. 알림 이동 URL 구조 설계
- **자기소개서 작성 URL:** `https://narratix/자기소개서/{자기소개서 ID}/{문항 번호}`
- **첨삭 URL (본인 접속):** `https://narratix/자기소개서/첨삭/{자기소개서 ID}/{문항 번호}?첨삭번호={첨삭 ID}`
- **첨삭 URL (첨삭자 접속):** `https://narratix/자기소개서/첨삭/{자기소개서 ID}/{문항 번호}?첨삭번호={첨삭 ID}&유효기간={유효 기간}`
- 본인 접속과 첨삭자 접속의 구분 방법에 대한 추가 논의 필요

### 2. 자료 업로드 TAB AI 에러 케이스
**AI 에러 케이스:**
- 업로드 실패: 문항 분류 실패, 파일 전체, 전체 내용 없는 경우

## 🛠️ Backend 개발 계획

프론트엔드 개발 속도에 맞춰 **기본적인 CRUD API 구현**에 집중합니다. 프론트엔드 작업이 지연되지 않도록 요청 순서에 따라 API를 우선 배포할 예정입니다.

### 1. 주요 기능 구현 (이번 주 목표)
* **검색 & 라이브러리**
  * 검색 API (문항 스크랩 검색)
  * 문항 라이브러리 전체 검색 구현
* **첨삭 시스템 (Correction)**
  * 첨삭 링크 활성화/비활성화 Toggle 기능
  * 링크가 활성화된 자기소개서 목록 조회
  * 첨삭 Websocket 구현 (**주말까지 완료 목표**)
* **인증 & 보안**
  * Refresh Token 로직 구현
* **자료 관리**
  * 자료 업로드 API 구현

## 🖥️ Frontend 개발 계획

### 1. UI 개발
* **목표:** 이번 주 내로 **모든 UI 작업 완료**

### 2. API 연동 (금요일까지)
첨삭 기능을 제외한 나머지 주요 도메인의 API 연결을 모두 완료합니다.

* **인증 (Auth)**
  * 회원가입, 로그인 연동
  * **보안 강화:** Access Token 관리 방식을 `In-memory`로 이관
  * **라우팅 가드:** 비로그인 상태에서의 URL 직접 접근 방지 처리
* **자기소개서 (Cover Letter)**

  * 나의 채용공고 연동
* **자료 및 라이브러리**
  * 자료 업로드 기능 연동
  * 라이브러리 페이지 연동
* **메인 & 기타**
  * 메인 화면 데이터 연동
